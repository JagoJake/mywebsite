C1 Software Development - Week 5 & 6

Week 5 & 6 things have really started to progress which has been amazing, we've been looking into file structure and its importance, DNS (Domain Name System), web hosting, SEO (search engine optimization) and VCS (Version Control Systems).

Git and Github have also been a big focus for week 5 & 6, understanding the terminal for Git and for GitHub creating an account, creating a repository, creating a branch, committing changes, using pull requests and merging. The Coding hasn't stopped either, I have now created a full portfolio website and hosted it with Github pages.

On the Career front, I have created a LinkedIn account and I have started networking.

C1 Software Development - Week 3 & 4

Week 3 & 4 have been really interesting, we've jumped into all things CSS (Cascading Style Sheets) discussing the pros and cons of external, internal and inline styles. We have gone through the basics like selectors, declarations, properties and values right through to more complex ideas like box model, grids and flexbox.

Bootstrap and SCSS are two things we have also been learning this past fornite, all the while using these skills to build responsive and engaging projects.

Learning as a community has been amazing, and seeing how others choose to build and design their pages and share ideas.

Great feedback from my career coach Alan regarding my CV, he suggested some changes which were very insightful and I implemented them (mainly to showcase how skills are transferable).

C1 Software Development - Week 1 & 2

Week 1 & 2 of studying with correlation one's software development program has been great, so far we've deep-dived into HTML, learning whilst building basic web page projects inside the Codio IDE (Integrated development environment) and getting to grips with all different types of tags, syntax, best practices and much more!

We’ve also taken a look at the differences between the world wide web and the Internet, two terms that are often used interchangeably but are far in fact two different things.

Learning as a community has been amazing, and seeing how others choose to build and design their pages and share ideas.

I've touched base with my career coach Alan which has been a great resource, we've spoken about where I am now and where I want to go and I've sent him my CV for review.
